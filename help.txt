Этот бот хранит вашем записи в зашифрованном виде
Записи защищены вашим мастер-паролем
Ваш мастер-пароль невозможно прочитать при закрытом хранилище
В случае утери мастер-пароля прочитать записи будет невозможно

Запись - это название сайта/ПО, а также логин и пароль от него.

Для управления записями есть три команды:
 > /add для добавления
 > /find для просмотра
 > /del для удаления
Для управления вашим хранилищем столько же:
 > /change - для смены мастер-пароля
 > /start - для открытия
 > /exit - для закрытия

При добавлении записи последовательно вводятся название сайта/ПО, логин и пароль.
Если ввод данных был прерван, введёные данные не сохраняются.

При просмотре и удалении записи, можно выбрать её в интерактивном списке или ввести фрагмент её названия вручную.

Записи могут содержать смайлики :)
